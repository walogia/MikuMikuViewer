# MikuMikuViewer
MMD用图片查看器
![](https://github.com/walogia/MikuMikuViewer/blob/master/%E9%A2%84%E8%A7%88.png)

## 使用方法：
1.拖拽可打开png,jpg,bmp,gif,sph,spa,tga,dss,psd格式图片文件

2.右键，可使用对话框打开图片，更改预设配色，最小化，退出程序
