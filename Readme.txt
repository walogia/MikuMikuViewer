【软件名】MikuMikuViewer

【著作者】吃爆米花的小熊

使用方法：

	1.拖拽可打开png,jpg,bmp,gif,sph,spa,tga,dss,psd格式图片文件
	2.右键，可使用对话框打开图片，更改预设配色，最小化，退出程序
